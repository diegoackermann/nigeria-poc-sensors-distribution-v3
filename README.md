# Nigeria POC Sensors Distribution — V3

Updated client-facing fullscreen map showing the Nigeria POC sensor distribution for Lagos: 15 recommended AllUnite sensors and modelled screen locations.

This is the V3 (updated client-facing) version. The V2 version is published from [nigeria-poc-sensors-distribution](https://github.com/diegoackermann/nigeria-poc-sensors-distribution).
